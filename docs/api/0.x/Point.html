Point

Represents a 2d point.
Can be created in serveral ways:

new Point(x, y) [source] 

var point = new Point(1000, 1000);
var point = new Point([1000,1000]);
var point = new Point({x:1000, y:1000});
Parameter Type Description
x Number x value
y Number y value
